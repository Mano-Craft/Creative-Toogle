# Creative Toggle – Fabric Mod

**Minecraft 1.21.0 + | Fabric**

## What does the mod do?

Press **F3 + X** (default) to switch to **Creative Mode** - just like `F3+N` switches to Spectator Mode.
Press **F3 + X** again to return to your **previous game mode**.

> Only works if the player has permission to use `/gamemode` (Singleplayer or OP on a server).

## Configuration

The key can be changed in the **Settings** under the **Creative Toggle** category:

* Via **Mod Menu** → Gear icon next to **Creative Toggle**
* Or directly in-game through the settings-key-screen

### Manually (config file)

`config/f3toggle.json`:

```json
{
  "toggleKeyCode": 88
}
```

GLFW key codes: X = 88, Z = 90, C = 67, V = 86, etc.

## Compatibility

| 1.21.0 + | ✅ |

The mod uses only stable Fabric API and Vanilla APIs, with no version-specific hacks.

# Creative Toggle – Fabric Mod

**Minecraft 1.21.0 + | Fabric**

## Was macht die Mod?

Drücke **F3 + X** (Standard), um in den **Kreativmodus** zu wechseln – genau wie `F3+N` in den Spectator-Modus schaltet.  
Drücke **F3 + X** nochmal, um in deinen **vorherigen Spielmodus** zurückzukehren.

> Funktioniert nur, wenn der Spieler die `/gamemode`-Berechtigung hat (Singleplayer oder OP auf Servern).

## Einstellungen

Die Taste kann in den **Einstellungen** unter dem Reiter **Creative Toggle** geändert werden:

- Über **Mod Menu** → Zahnrad-Symbol neben „Creative Toggle"
- Oder direkt im Spiel über den Einstellung-Steuerung-Bildschirm

### Manuell (config-Datei)
`config/f3toggle.json`:
```json
{
  "toggleKeyCode": 88
}
```
GLFW-Key-Codes: X = 88, Z = 90, C = 67, V = 86, usw.

Die Mod nutzt ausschließlich stabile Fabric-API und Vanilla-APIs, keine versionsspezifischen Hacks.
